# 游戏伙伴

一个 AstrBot 群聊插件，可为任意游戏建立独立的伙伴名单。例如：

- `/加入农伙伴`
- `/加入原神伙伴`
- `/加入LOL伙伴`
- `/召集原神伙伴`

## 指令

| 指令 | 说明 |
| --- | --- |
| `/伙伴帮助` | 显示群内教程 |
| `/加入<游戏>伙伴` | 将自己加入名单 |
| `/邀请加入<游戏>伙伴 @群友` | 邀请一人或多人 |
| `/同意加入<游戏>伙伴` | 接受未过期的邀请 |
| `/拒绝加入<游戏>伙伴` | 拒绝邀请 |
| `/退出<游戏>伙伴` | 将自己移出名单 |
| `/召集<游戏>伙伴` | 一键 @ 所有伙伴（只有该伙伴成员可用） |
| `/查看<游戏>伙伴` | 查看并 @ 当前成员 |
| `/伙伴列表` | 显示本群所有游戏及人数 |
| `/我的伙伴` | 显示自己已加入的游戏 |

## 配置

安装后可在 AstrBot WebUI 的插件配置中调整：

- 召集冷却秒数，以及按“游戏全局”或“召集人”计算
- 管理员是否跳过冷却
- 管理员是否可在未加入名单时召集
- 是否允许自助加入、是否允许邀请
- 邀请是否要求发起人已在名单、邀请有效期
- 游戏名最大长度

## 数据兼容

v1 版本的 `nong_partners:<群号>` 数据会在第一次使用时自动迁移为“农伙伴”，原数据不会被删除。新数据使用 AstrBot 插件 KV 存储，按群隔离。

## 安装

在 AstrBot 插件管理中通过 GitHub 仓库地址安装，或将本仓库放入 `data/plugins/astrbot_plugin_nong` 后重载插件。要求 AstrBot `>=4.9.2,<5`。
